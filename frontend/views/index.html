What it does

Built for real-world traffic

Every feature designed to reduce wait times and keep traffic flowing.

AI camera detection

YOLO model counts vehicles at each junction in real time, sending live density data to the system every second.

Adaptive signal control

ESP32 controllers calculate optimal green light durations each cycle based on live vehicle counts per lane.

Emergency override

Force all red, N-S green, or E-W green instantly from the dashboard. Duration-controlled with auto-revert.

Live analytics

Traffic flow charts, per-junction throughput, wait time breakdowns and system health on a dedicated page.

ESP32 wireless control

Two ESP32 boards communicate over WiFi — receiving commands and reporting decisions back in real time.

Traffic simulation

Run controlled simulations to test signal timing strategies before deploying changes to live junctions.
